## 📖 Sobre o Projeto
O portfólio foi desenvolvido como parte do desafio da DIO, com foco em:
- Criar uma página responsiva e moderna.
- Apresentar informações pessoais e profissionais.
- Exibir projetos realizados e links para repositórios.
- Demonstrar domínio em HTML, CSS e JavaScript.

---

## 🛠 Tecnologias Utilizadas
- **HTML5** → Estrutura semântica da página.  
- **CSS3** → Estilização e responsividade.  
- **JavaScript** → Interatividade e dinamismo.  
- **Git/GitHub** → Controle de versão e hospedagem.  

---

## ✨ Funcionalidades
- Página inicial com apresentação pessoal.  
- Seção de projetos com links para GitHub.  
- Seção de habilidades e tecnologias dominadas.  
- Formulário de contato funcional.  
- Layout responsivo para diferentes dispositivos.
